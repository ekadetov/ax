# ax Workflow Plugin

AI development workflow skills for Claude Code.

## Pipeline

write-a-prd -> prd-to-issues -> issue-to-tasks -> (code) -> code-review -> final-audit

## Skills

- `write-a-prd` - Create a PRD via user interview and codebase exploration
- `prd-to-issues` - Break a PRD into vertical-slice issues
- `issue-to-tasks` - Break an issue into session-sized AI tasks
- `code-review` - Six-pass code review (logic, ordering, practices, security, magic values, patterns)
- `final-audit` - Cross-cutting post-merge audit across all feature files
- `grill` - Adversarially stress-test a plan by challenging every assumption
- `ubiquitous-language` - Extract a DDD-style glossary from conversation
- `design-interface` - Generate multiple radically different interface designs (Design it Twice)
- `improve-architecture` - Explore codebase for deep-module opportunities and file GitHub issue RFCs
